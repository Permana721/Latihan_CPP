## 📚 Materi Repository
Berikut adalah daftar materi dan struktur data yang telah diimplementasikan dalam repositori ini:

1.  **Struct (`_1_Struct`)**: Penggunaan tipe data bentukan untuk mengelompokkan berbagai jenis data dalam satu kesatuan.
2.  **Pointer (`_2_Pointer`)**: Pemahaman tentang alamat memori, referensi, dan alokasi memori dinamis.
3.  **Stack (`_3_Stack`)**: Implementasi struktur data LIFO (*Last In First Out*) menggunakan array atau linked list.
4.  **Queue (`_4_Queue`)**: Implementasi struktur data FIFO (*First In First Out*) untuk antrean.
5.  **Tree (`_5_Tree`)**: Struktur data non-linear hierarkis, termasuk *Binary Tree* dan *Binary Search Tree* (BST).
6.  **Graph (`_6_Graph`)**: Representasi hubungan antar objek menggunakan *Adjacency Matrix* atau *List*.
7.  **Sorting (`_7_Sorting`)**: Kumpulan algoritma pengurutan data (seperti Bubble, Selection, Insertion, Quick, atau Merge Sort).
8.  **Searching (`_8_Searching`)**: Algoritma pencarian data seperti *Linear Search* dan *Binary Search*.

## 🎯 Tujuan Pembelajaran
* Memahami konsep manajemen memori melalui Pointer.
* Menguasai implementasi struktur data dasar (Stack, Queue, Tree, Graph).
* Mampu menganalisis dan menerapkan algoritma pengurutan serta pencarian yang efisien.
